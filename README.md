# Simple FastAPI Auth System

A lightweight web application featuring a registration and login system. Built with Python's FastAPI, this project demonstrates secure password handling, database integration, and dynamic frontend error messaging.

## 🚀 Features

- **Backend:** Fast and modern API built with FastAPI.
- **Security:** Password hashing using `bcrypt` to ensure user data safety.
- **Database:** Local user storage using SQLite and SQLAlchemy ORM.
- **Frontend:** Interactive UI with dynamic error notifications and automatic URL cleanup (using JavaScript's `history.replaceState`).
- **Styling:** Modern, dark-themed CSS cards with responsive inputs.

## 🛠 Tech Stack

* **Python 3.9+**
* **FastAPI** (Web framework)
* **SQLAlchemy** (Database ORM)
* **Bcrypt** (Password encryption)
* **SQLite** (Database)
* **JavaScript/HTML/CSS** (Frontend)
